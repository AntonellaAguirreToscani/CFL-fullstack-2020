Que es Nest?

Nest es un Framework para crear aplicaciones de servidor Node.js escalables y eficientes.

Combina elementos de OOP (Programación orientada a objetos) y FRP.

Nest proporciona un nivel de abstracción por encima de estos marcos comunes de Node.js (Express / Fastify),

pero también expone sus API directamente al desarrollador. Esto les da a los desarrolladores la libertad de usar

la gran cantidad de módulos de terceros que están disponibles para la plataforma subyacente.
